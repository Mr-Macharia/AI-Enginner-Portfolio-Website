# Gichogu Macharia — Portfolio

Personal portfolio website for Gichogu Macharia, AI/ML Engineer based in Nairobi, Kenya.

Live at [gichogumacharia.tech](https://gichogumacharia.tech)

---

## About

This site showcases my work in AI/ML engineering, data analytics, full-stack development, and business automation. It includes project highlights, professional experience, certifications, skills, and a working contact form.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 + TypeScript |
| Build Tool | Vite |
| Styling | Custom CSS + Tailwind CSS |
| Animations | Framer Motion |
| Contact Form | EmailJS |
| Hosting | Vercel |

---

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Production build
npm run build
```
